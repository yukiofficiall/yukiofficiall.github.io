  • 太原理工大学


    太原理工大学是一所历史悠久、底蕴深厚、特色鲜明的世纪学府， 坐落于具有2500多年建城史的国家历史文化名城——太原。120年来， 学校始终秉承「求实、创新」的校训，坚持「以人为本、文体为舟、承载德智、全面发展」的办学传统， 彰显「敢为人先、敢于竞争、勇于创新」的精神气质，涌现出一批学术大师、行业翘楚和道德楷模。

  • 电气与动力工程学院


    电气与动力工程学院始建于1918年山西大学堂西学专斋时开设的电气学门。在办学历程中，学院继承和发扬了“敬业、尚德、笃行、博学、创新”之精神，坚持“以立德树人为根本，以学生发展为中心”的培养目标；以学科建设为龙头，以服务地方经济建设为出发点，以扎实的作风，坚韧不拔的意志品质和勇于创新的精神，造就了一个又一个的辉煌，为山西乃至全国的经济建设做出了突出的贡献。

    eRF  （神经辐射场）
    Neural Radiance Fields
    神经辐射场（Neural Radiance Fields，NeRF），是一种基于深度学习的三维重建技术。该技术最初由Mildenhall等人提出，并在2020年得到了广泛的关注。与传统的三维重建技术不同，NeRF并不依赖于已知的三维模型或场景几何形状，而是通过学习复合函数来描述三维空间中点处的辐射场。因此，即使处理复杂的场景，使用NeRF依然能够快速、精确地进行三维重建。

    NeRF is a type of 3D reconstruction technique based on deep learning. Initially proposed by Mildenhall, it gained widespread attention in 2020. Unlike traditional 3D reconstruction techniques, NeRF does not rely on known 3D models or scene geometry, but instead uses a learned composite function to describe the radiance field at each point in 3D space.
